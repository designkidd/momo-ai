9. Dark Mode Rules 深色模式規則

  • Theme attr: data-theme="dark" on <html>
  • Borderless at rest: Form elements use border-color: transparent, shown on hover/focus
  • Focus rings: Semi-transparent rgba(), never solid
  • :focus vs :focus-visible: :focus = subtle border, :focus-visible = glow
  • Hover surface: #383a3d (not --bg-card)
  • Active hover: Always brighten, never darken + add glow
  • Native select: Replace with custom component (.csel-*)
  • color-scheme: Set color-scheme: dark on :root
